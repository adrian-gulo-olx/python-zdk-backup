Мой профиль

Личная учетная запись Мой профиль используется для управления вашими объявлениями. Доступ к личной учетной записи дает вам следующие возможности:

  • мгновенное размещение объявления без подтверждения (если вы авторизированы);
  • внесение любых изменений в объявления;
  • удаление неактуальных объявлений;
  • обновление объявлений по дате каждые 7 дней;
  • просмотр статистики по каждому объявлению;
  • получение и отправка личных сообщений;
  • просмотр истории платежей и счета OLX;
  • изменение настроек контактных данных, пароля, электронного адреса, электронных сообщений, рассылки;
  • настройка SMS-сообщений о новых ответах на ваши объявления, полученные по электронной почте, для Топ-объявлений;
  • удаление учетной записи;
  • оформление доставки.

Мой профиль содержит следующие разделы:

1. Объявления - Активные (доступные на сайте), Ожидающие (объявления, которые проходят проверку модератором), Неактивные (удаленные вами объявления), Удаленные (объявления, удаленные модератором).

Действия с Активными объявлениями:

- Просмотреть: просмотреть страницу объявления на OLX.

- Редактировать: изменить параметры объявления, добавить фотографии.

- Деактивировать: удалить объявление в список «Неактивные объявления».

- Сообщения: просмотреть сообщения по объявлению.

- Рекламировать: повысить эффективность объявления, заказав дополнительные услуги (Топ-объявление, Срочно, Выделение, Поднятие).

- Статистика: количество просмотров вашего объявления, просмотров полей Телефон, ICQ, Skype. Доступна функция обнуления просмотров объявления. Количество добавлений объявления в Избранные другими пользователями.

- Vkontakte, Одноклассники, Facebook, Twitter: разместить объявление в социальных сетях.

- Листовка: распечатать объявление в виде листовки.

- Виджет: разместить виджет на сайте или в блоге.

- Обновить: бесплатно обновить объявление по дате (эта кнопка доступна 1 раз в 7 дней).

Действия с Ожидающими объявлениями:

- Просмотреть: просмотреть страницу объявления, какой она будет на сайте после проверки модератором.

- Редактировать: изменить параметры объявления, добавить фотографии.

- Отменить публикацию: отменить размещение объявления, при этом оно будет окончательно удалено, без возможности восстановления.

Действия с Неактивными объявлениями:

- Просмотреть: просмотреть страницу объявления в Архиве сайта.

- Редактировать: изменить параметры объявления, добавить фотографии.

- Удалить: удалить объявление окончательно, без возможности восстановления.

- Сообщения: просмотреть сообщения по объявлению.

- Активировать: восстановить ранее удаленное объявление в список «Активные объявления».

Действия с Удаленными объявлениями:

- Просмотреть: просмотреть страницу объявления.

- Удалить: удалить объявление окончательно, без возможности восстановления.

- Узнать подробнее: просмотреть сервисное сообщение о причине удаления объявления модератором. Если объявление отправлено на редактирование, вы можете внести требуемые исправления и восстановить объявление с помощью кнопки Исправить в тексте сервисного сообщения. Если кнопка Исправить недоступна - объявление не может быть восстановлено.

Примечание: если у вас нет пароля (вы незарегистрированный пользователь), то вы сможете управлять своим объявлением с помощью ссылок, указанных в письме-оповещении, отправленном на ваш электронный адрес после публикации объявления.

Для поиска необходимых объявлений в учетной записи доступна функция Поиск.

Поиск можно осуществлять во всех списках объявлений (Активные, Ожидающие, Неактивные, Удаленные) по ключевому слову, которое используется в заголовке, или номеру объявления.

2. Сообщения - в данном разделе доступна переписка по вашим объявлениям, а также объявлениям других пользователей, с которыми вы инициировали переписку. В разделе сохраняются Полученные, Отправленные и Архивные сообщения.

- Полученные и отправленные сообщения: входящие и исходящие сообщения по вашим объявлениям, а также по объявлениям других пользователей, с которыми вы ведете переписку. Данные сообщения можно Отметить звездочкой, а также переместить в Архив. Также, в данном разделе доступен фильтр по Непрочитанным и Отмеченным звездочкой сообщениям.

- Архивные сообщения: архив удаленных вами входящих/исходящих сообщений. Доступно восстановление, а также окончательное удаление сообщений из Архива.

3. Платежи и счет OLX.ua - в данном разделе хранится история заказов дополнительных услуг, а также указан баланс вашего счёта OLX.ua.

4. Настройки - раздел, в котором вы можете внести изменения в настройки вашей учетной записи.

- Изменить контактные данные: в данном разделе вы можете изменить контактные данные, которые вы указали при размещении объявления. Также, вам доступна функция включения/отключения предзаполнения таких полей как Регион, Город, Точный адрес, Контактное лицо, Номер телефона, Skype, ICQ, а также функции отображения ссылки на все ваши объявления на странице вашего объявления.

- Изменить пароль: в данном разделе вы можете создать новый пароль, который вы используете для авторизации.

- Изменить email-адрес: вы можете изменить электронный адрес, который используете для входа на OLX.ua.

- Email-сообщения : вы можете включить/выключить рассылку новостей, а также email-сообщений о новых ответах.

- SMS-уведомления: здесь вы можете включить уведомления по SMS о новых ответах на ваши объявления, полученных по электронной почте. SMS-уведомления работают только для Топ-объявлений.

- Удалить учетную запись: если размещение объявлений для вас более неактуально, используйте кнопку Удалить учетную запись.
