Conselhos para uma transacção bem-sucedida

O que terá de se lembrar quando finalizar a transacção:

A finalização das transacções é importante, por isso deveria ter em conta algumas coisas:

Local

Tente escolher locais públicos, evitar locais sem testemunhas. Se não conhece a pessoa com quem quer fazer negócio, seja cauteloso.

Item

Verifique se o produto ou serviço está exactamente como descrito no anúncio, ou se está danificado ou defeituoso e, se possível, se funciona como deveria.

Pagamento

Utilize o método mais fácil de pagamento: dinheiro. Contudo, se um produto ou serviço é muito dispendioso, o utilizador poderá requisitar o pagamento através de transferência bancária.

Dados pessoais

Não faculte demasiada informação pessoal. Lembre-se que apenas se encontra com o vendedor para finalizar a transacção.

Caso tenha dúvidas, Não hesite em contactar o vendedor colocando as suas questões de forma a ficar esclarecido antes de efectuar a transacção.
