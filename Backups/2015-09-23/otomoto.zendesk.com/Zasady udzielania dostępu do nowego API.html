Zasady udzielania dostępu do nowego API

  1. Dostęp do nowego API jest udzielany firmom, które zajmują się obsługą informatyczną dealerów oraz tworzą oprogramowanie umożliwiające dodawanie ogłoszeń do serwisu otomoto i ich pobieranie na strony www dealerów.
  2. Do każdej aplikacji tworzonej przez wspomniane firmy jest udzielany wyłącznie 1 klucz API, niezależnie od tego ilu dealerów dana aplikacja obsługuje. Oznacza to, że wielu dealerów może korzystać z tego samego klucza, ktróy jednak powinien być zakodowany w aplikacji, a nie udzielany dealerom bezpośrednio.
  3. W celu uzyskania klucza należy napisać na adres api@otomoto.pl, podając nazwę firmy, dane osoby kontaktowej oraz liczbę obsługiwanych dealerów.
  4. O przyznaniu klucza każdorazowo decyduje otomoto. W przypadku decyzji pozytywnej, nowy klucz wraz z jego ID są przesyłane mailowo.
