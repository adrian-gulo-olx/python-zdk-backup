How can I post an ad without logging in?

You may do so by:

1. Clicking the ‘Sell your item now’ button from the homepage
2. Completely filling out all the required ad details
3. By pressing submit, you should be able to see this ad confirmation page

 

4. Enter the code that you received via SMS and input it in the code field and click ‘Confirm Ad’
