Am I allowed to post the same ad in different countries / cities / languages ?

Below are the rules for posting the same ad in:

  • Different Countries:

You can only post an ad in the same country that the item is available in, so that you can deal with the buyer face to face. 

  • Different Cities:

You can post the same ad in 3 different cities in the same country, so you can attract more buyers.

  • Different Languages:

You can post the same ad in 2 different languages (Arabic and English).
