Pobranie listy dzielnic danej miejscowości

Api umożliwia pobranie listy dzielnic danej miejscowości, która posiada dzielnice. Służy do tego parametr district_city_id, który jednak nie jest tożsamy z city id. Pobierając listę miejscowości otrzymujemy następujące dane miejscowości:

{
"id": 627,
"text": "Aleksandrów Kujawski, aleksandrowski, Kujawsko-pomorskie",
"lon": 18.69801,
"lat": 52.86653,
"radius": 3,
"zoom": 13,
"name": "Aleksandrów Kujawski",
"url": "aleksandrow-kujawski",
"districts_city_id": 2,
"region_id": 15
},

Aby pobrać listę dzielnic dla tej miejscowości należy użyć nie id miejscowości, lecz district_city_id. Poniżej przykład dla Katowic.

GET https://ssl.otomoto.pl/api/open/districts?city_id=13155

Authorization  Bearer 70bd5f2620dedd816cccb55ce98b96c3b87351ad

Accept application/json
