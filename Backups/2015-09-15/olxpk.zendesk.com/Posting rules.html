Posting rules

The site OLX.com.pk has some principles/rules to be followed:

  1. Post only one ad for an item. If you want to post an ad with the same details as last time, then do not add new advertisement - just activate the previously added advertisement.
  2. Always make sure that your item or service is not subject to Prohibition.
  3. Take care of the exact description. Particularly in the case of job announcements.
  4. If you see any violation of the rules - report them to us.
